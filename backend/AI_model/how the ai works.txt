the main.py is the starting point of the ai process, this file will require an image, after that main will call the ai model 'best.pt', the model will predict the landmarks and return an image with the landmarks and a txt file that contains the coordination's of each point with its class id. After that a message will appear that asks if the user want's a steiner analysis type, after ok is clicked, main will call steiner_analysis.py file that is responsible of applying steiner analysis to the output of the model and store the result in a path that can be seen in 'steiner_output_dir' variable
